This is a ecommerce website (fabrica) using mern stack.
Fabrica is a modern, responsive e-commerce platform built using the MERN stack (MongoDB, Express.js, React.js, Node.js). Designed to streamline online fabric and clothing shopping, the app allows users to explore products, select sizes and quantities, and manage their cart with a seamless user experience.

Key Features:

 Product Listings with images, size & quantity options

Add to Cart & Order Summary functionality

User Authentication (JWT-based)

Admin Panel for uploading products with preview support

Responsive design (mobile & desktop)

Tech Stack:

Frontend: React.js with Tailwind CSS for modern UI

Backend: Node.js + Express.js for RESTful APIs

Database: MongoDB for data storage

Other Tools: Cloudinary for image storage, FormData for image upload handling, and Postman for API testing
